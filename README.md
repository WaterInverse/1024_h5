# 月光生活家 · 移动端 H5 记账（样例）

快速启动：
1. npm install
2. npm run dev
3. 用手机扫码或在浏览器切换到移动设备视图测试

主要功能：
- 快速记账（类别、金额、日期默认当天、备注、上传照片）
- 时间线卡片展示（卡片含图标、类别、金额、日期、备注）
- 月度饼图/趋势图（ECharts）
- 预算管理（进度条提示）
- 点击卡片查看详情（备注、照片）
- 每月底生成趣味标签与简短总结

开发提示：
- UI 优先移动端体验，设置 viewport 并采用 rem 或 flexible 布局
- 图片使用 base64 / 压缩后存储，避免大文件影响性能
- 先以 localStorage 为持久层，后续可接入后端同步
